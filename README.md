# Loopwish · Web

![Loopwish banner](https://raw.githubusercontent.com/loopwish/shared/main/assets/logos/loopwish-banner.svg)

Loopwish er en ønskeliste-app som lar deg dele ønsker med familie og venner. Ønsk. Del. Få. Sammen.

Dette repoet inneholder web-klienten for Loopwish.

## Teknologi-stack

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- ESLint + Prettier
- Vitest

## Kom i gang

### Forutsetninger

- Node.js 20+
- npm

### Installering

```bash
git clone https://github.com/loopwish/web.git
cd web
npm install
```

### Kjør lokalt

```bash
npm run dev
```

Åpne http://localhost:3000

### Kvalitetssjekker

```bash
npm run lint
npm run format:check
npm test
npm run build
```
